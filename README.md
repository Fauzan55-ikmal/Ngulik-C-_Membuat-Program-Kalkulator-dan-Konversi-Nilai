# Program Kalkulator & Konversi Nilai

Proyek C++ modular dengan struktur CMake.

## Cara Menjalankan

1. Pastikan CMake sudah terinstall.
2. Buka terminal di folder proyek.
3. Jalankan `mkdir build && cd build`
4. Jalankan Program Kalkulator & Konversi Nilai
   Proyek C++ modular dengan struktur CMake yang mencakup fitur kalkulator, konversi nilai, dan sorting.

Fitur
Kalkulator: Operasi dasar matematika.

Konversi Nilai: Mengonversi nilai angka ke huruf.

Sorting: Mengurutkan angka secara otomatis.

Cara Menjalankan
Pastikan CMake sudah terinstall.

Buka terminal di folder proyek.

Jalankan mkdir build && cd build

Jalankan cmake -G "MinGW Makefiles" .. lalu cmake --build .

Jalankan ./ProgramUtama.exe`cmake ..` lalu `make` (atau build via VS Code). 5. Jalankan `./ProgramUtama`
